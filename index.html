Lesson 1

The Internet

The web is a bunch of computers that talk with each other through html. When you go to particular site the computer sends a HTTP Request to a server.

Say you go to here to check the scores for the NHL playoffs .Your computer will send a request to a server and it will find the right http.

HTML

HTML stands for Hypertext Markup Language. HTML documents basically make up just about every web page on the internet.

Tags and Elements

HTML documents are made of HTML elements.

When writing HTML, the programmer will tell browsers the type of each element by using HTML tags. Some examples are italicize, bold.

Why Computers are Stupid

Computers are stupid because they follow direction too well. If you forget an entag on something, it could mess up the whole document

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an empty white space around the element.

Lesson 2

The "tree-like structure" of HTML

HTML files are the structure of a page. If you go into the developer tools, you can see a tree like structure. A pretty good example of how to describe it would be a family tree.

In a browser, this structure appears as boxes. There are boxes inside of boxes inside of boxes. EVERYTHING IS MADE OF BOXES!

The relationship between indented HTML and boxes

When you read an HTML document as text, you see a bunch of different indentations going up and down the left side of the document. The more indented an element is, the more deeply inserted its box is going to be.
